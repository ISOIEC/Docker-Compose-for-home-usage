# Docker-Compose
Файлы для создания Staks в Porainer [AdguardHome](https://github.com/ISOIEC/Docker-Compose-for-home-usage/blob/main/adguard.yaml), Plex, qBittorrent, [Planka](https://github.com/ISOIEC/Docker-Compose-for-home-usage/blob/main/planka.yaml), [Homarr](https://github.com/ISOIEC/Docker-Compose-for-home-usage/blob/main/homarr.yaml)  
Подразумевается что у вас уже стоит [Docker](https://docs.docker.com/desktop/setup/install/linux) и [Portainer](https://docs.portainer.io/start/install-ce/server/docker/linux)  
Файлы сделаны для себя, для облегения переустановки или перенастройки в будущем
